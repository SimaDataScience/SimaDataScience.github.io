NFL Web Scraping with Python and Visualizaiton with RShiny

How To Use This Dashboard

This dashboard was created using the score of every NFL game since 2002. It can be used to summarise a single team's previous performance, compare how two teams have performed against each other historically, or show team rankings by year.

The controls at the top of each page control which team/s to look at, which metric to use, and how far back to go when calculating statistics. Click on the football field above to be taken to the dashboard!

Note: In all visualizations and calculated win/loss statistics, zero corresponds to a loss, one corresponds to a win, and one half corresponds to a tie.

Where Did I Get My Data?

The story of this project was the story of data collection. Originally, I wanted to create an extensive dashboard with detailed player and team statistics. However, the more I looked the more I realized that these statistics were almost unanimously either located behind a paywall or not allowed to be scraped. I had two options: I could use IP rotation / random interval resting to get the data anyway, or I could do the ethical thing and get creative with the data that I was allowed to scrape. I chose the later, and obtained only game score data from Pro Football Reference. The dashboard shows my attempt to get as much information as possible from only team names and team scores.

I created a notebook to scrape Pro Football Reference using Selenium and Beautiful soup, which can be found here. An oversight almost caused me to miss all games that resulted in ties, which is a great reminder of the importance of data validation. I include the ties manually and also perform some basic data cleaning in python before the data is then ready for visualization in R.

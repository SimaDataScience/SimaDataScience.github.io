Implementing and Training a GPT-3 Inspired Transformer

Here I implement a smaller (and slightly tweaked) version of GPT-3 using Pytorch, inspired by Andrej Karpathy's [nanoGPT](https://github.com/karpathy/nanoGPT). The script can be used to specify model size / training arguments and train a GPT-3 like model on a given text file/s. Changes to original: Character-level encoding is utilized instead of BPE in order to keep the vocabulary size small. Multi-head attention blocks are not parallelized as efficiently (for now...). ReLU activation is used instead of GeLU.
